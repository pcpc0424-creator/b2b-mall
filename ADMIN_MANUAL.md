## 1. 대시보드

### 접속 경로
`/admin`

### 주요 기능
실시간 쇼핑몰 현황을 한눈에 확인할 수 있습니다.

### 표시 정보

| 구분 | 설명 |
|------|------|
| **총 상품** | 등록된 전체 상품 수 |
| **총 회원** | 탈퇴 회원 제외한 회원 수 |
| **총 주문** | 전체 주문 건수 |
| **총 매출** | 누적 매출액 |
| **오늘 주문** | 오늘 들어온 주문 건수 |
| **오늘 매출** | 오늘 매출액 |
| **대기 주문** | 결제 대기 중인 주문 건수 |
| **재고부족** | 재고 부족/품절 상품 수 |
| **활성 프로모션** | 현재 진행 중인 프로모션 수 |
| **활성 회원** | 활성 상태 회원 수 |

### 빠른 작업
- 상품 등록
- 주문 처리
- 회원 관리
- 프로모션 관리
- 배송 설정

## 2. 상품 관리

### 접속 경로
`/admin/products`

### 주요 기능

#### 상품 목록
- **검색**: 상품명, SKU로 검색
- **필터**: 카테고리, 세부 카테고리, 재고 상태별 필터링
- **가격 표시**: 정상가(취소선) + 판매가(굵게) 함께 표시

#### 상품 등록/수정 (`/admin/products/new` 또는 `/admin/products/{id}/edit`)

| 항목 | 설명 |
|------|------|
| **기본 정보** | 상품명, SKU, 브랜드 |
| **카테고리** | 대분류 > 소분류 선택 |
| **가격 설정** | 정상가(retail), 회원가(member), 원가(wholesale) |
| **이미지** | 드래그&드롭 또는 클릭으로 이미지 업로드 (최대 5MB) |
| **재고 상태** | 충분(available), 부족(low), 품절(out_of_stock) |
| **최대 구매수량** | 1인당 최대 구매 가능 수량 (0 = 제한 없음) |
| **상품 설명** | 상세 설명 텍스트 |

### 재고 상태 표시
- 🟢 **충분**: 재고 여유
- 🟡 **부족**: 재고 부족 경고
- 🔴 **품절**: 재고 없음

## 3. 프로모션 관리

### 접속 경로
`/admin/promotions`

### 주요 기능

#### 프로모션 목록
- 검색 및 유형/상태 필터링
- 토글 스위치로 활성화/비활성화

#### 프로모션 등록/수정

| 항목 | 설명 |
|------|------|

| **설명** | 프로모션 설명 텍스트 |
| **이미지** | 배너 이미지 업로드 |

| **유형** | 전체 / 타임특가 / 전용 |
| **기간** | 시작일 ~ 종료일 |
| **대상 등급** | 비회원, 일반회원, 우수회원, VIP회원 선택 |

- **전체**: 모든 등급 대상

- **전용**: 특정 등급 전용

## 4. 주문 관리

### 접속 경로
`/admin/orders`

### 주요 기능

#### 주문 목록
- 주문번호, 고객명으로 검색
- 주문 상태별 필터링

#### 주문 상태

| 상태 | 설명 |
|------|------|
| **대기** (pending) | 결제 대기 중 |
| **확인** (confirmed) | 결제 완료, 주문 확인 |
| **준비** (preparing) | 상품 준비 중 |
| **배송** (shipped) | 배송 중 |
| **완료** (delivered) | 배송 완료 |
| **취소** (cancelled) | 주문 취소됨 |
| **환불** (refunded) | 환불 완료 |

#### 주문 상세 (눈 아이콘 클릭)
- **주문 상태 변경**: 드롭다운으로 상태 변경
- **운송장 정보 입력**:
  - 택배사 선택 (CJ대한통운, 한진, 롯데, 로젠, 우체국 등)
  - 운송장번호 입력
- **고객 정보**: 이름, 이메일, 회사
- **배송 정보**: 수령인, 연락처, 주소, 배송메모
- **주문 상품**: 상품명, 수량, 금액
- **결제 정보**: 상품금액, 배송비, 총 결제금액
- **환불 처리**: 결제 완료 상태인 주문에 환불 버튼 표시

### 환불 처리
1. 주문 상세에서 "환불 처리" 버튼 클릭
2. 확인 팝업에서 확인
3. 토스페이먼츠 API로 자동 환불 처리
4. 주문 상태가 "환불"로 변경됨

> ⚠️ 환불은 `paymentKey`가 있는 주문만 가능합니다.

## 5. 회원 관리

### 접속 경로
`/admin/members`

### 주요 기능

#### 회원 목록
- 이름, 이메일, 회사명으로 검색
- 등급, 상태별 필터링
- **회원 목록 / 탈퇴 회원** 탭 전환

#### 회원 정보 표시
- 이름, 가입 경로 (카카오/네이버/구글/이메일)
- 등급 배지 (비회원/일반회원/우수회원/VIP회원)
- 상태 배지 (활성/비활성/정지/승인대기/탈퇴)
- 소속, 추천인 정보
- 총 구매금액

#### 가입 승인
- 승인 대기 알림이 상단에 표시됨
- "승인" 버튼으로 빠른 승인 가능

#### 회원 상세 (눈 아이콘 클릭)
- **기본 정보**: 이름, 이메일, 전화번호, 가입경로, 회사, 사업자번호, 소속, 추천인, 마케팅 수신 동의
- **회원 등급 변경**: 버튼 클릭으로 등급 변경
- **회원 상태 변경**: 버튼 클릭으로 상태 변경
- **활동 정보**: 총 주문, 누적금액, 가입일, 최근주문
- **배송지 목록**: 회원이 등록한 배송지 목록

#### 탈퇴 회원 관리
- 탈퇴 회원 탭에서 확인
- "복구" 버튼으로 회원 복구 가능

## 10. 쿠폰 관리

### 접속 경로
`/admin/coupons`

### 주요 기능

#### 쿠폰 목록
- 전체/활성/비활성 필터링
- 눈 아이콘으로 활성/비활성 토글

#### 쿠폰 등록/수정

| 항목 | 설명 |
|------|------|
| **쿠폰 코드** | 사용자가 입력할 코드 (자동 대문자 변환) |
| **쿠폰 이름** | 쿠폰 이름 |
| **설명** | 쿠폰 설명 (선택) |
| **할인 유형** | 퍼센트 할인 (%) / 정액 할인 (원) |
| **할인율/금액** | 할인 값 |
| **최소 주문 금액** | 쿠폰 사용 가능한 최소 주문 금액 (선택) |
| **최대 할인 금액** | 퍼센트 할인 시 최대 할인 한도 (선택) |
| **시작일 / 종료일** | 쿠폰 유효 기간 |
| **활성화** | 체크 시 사용자가 코드로 등록 가능 |

#### 쿠폰 목록 아이콘 설명

| 아이콘 | 기능 |
|--------|------|
| 📋 (목록) | 발급 현황 보기 |
| ✈️ (비행기) | 회원에게 쿠폰 발급 |
| ✏️ (연필) | 쿠폰 정보 수정 |
| 🗑️ (휴지통) | 쿠폰 삭제 |
| 👁️ (눈) | 활성/비활성 토글 |

### 쿠폰 발급하기

회원에게 쿠폰을 직접 발급합니다.

1. 쿠폰 목록에서 **발급 아이콘(✈️)** 클릭
2. 발급 모달에서:
   - **전체 발급**: "전체 회원에게 발급" 버튼 클릭
   - **선택 발급**:
     - 이름/이메일/전화번호로 회원 검색
     - 체크박스로 발급 대상 선택
     - "N명에게 발급" 버튼 클릭
3. 결과 확인: 성공 N명, 이미 보유 N명

> 💡 이미 해당 쿠폰을 보유한 회원은 중복 발급되지 않습니다.

### 발급 현황 확인하기

누가 어떤 쿠폰을 보유하고 있는지 확인합니다.

1. 쿠폰 목록에서 **목록 아이콘(📋)** 클릭
2. 발급 현황 모달에서:
   - **요약 정보**: 총 발급, 사용완료, 미사용 건수
   - **검색**: 이름, 이메일로 회원 검색
   - **목록**: 회원별 발급일, 사용 상태, 사용일 확인

#### 발급 현황 테이블

| 항목 | 설명 |
|------|------|
| **회원** | 회원 이름, 이메일 |
| **발급일** | 쿠폰 발급 날짜 |
| **상태** | 미사용 (초록) / 사용완료 (회색) |
| **관리** | 회수 버튼 (미사용만 가능) |

### 쿠폰 회수하기

발급된 쿠폰을 회수합니다. **미사용 쿠폰만 회수 가능**합니다.

1. 발급 현황 모달에서 회수할 회원 찾기
2. **회수 아이콘(⊗)** 클릭
3. 확인 팝업에서 "확인" 클릭
4. 해당 회원의 쿠폰함에서 쿠폰이 제거됨

> ⚠️ 이미 사용된 쿠폰은 회수할 수 없습니다.

### 쿠폰 상태 배지

| 배지 | 설명 |
|------|------|
| **만료** (빨강) | 유효기간이 지난 쿠폰 |
| **대기** (노랑) | 아직 시작일이 되지 않은 쿠폰 |

| **베스트연구실** | 인기 상품 섹션 |
| **신상품연구실** | 신상품 섹션 |
| **초특가연구실** | 특가 상품 섹션 |

#### 상품 추가
1. "상품 추가" 버튼 클릭
2. 상품명/SKU로 검색
3. 원하는 상품 클릭하여 추가

#### 순서 변경
- ▲/▼ 버튼으로 상품 순서 조정

#### 상품 제거
- 휴지통 아이콘으로 섹션에서 제거

> ⚠️ 각 섹션에 최대 10개 상품만 등록 가능합니다.

## 14. 등급 설정

### 접속 경로
`/admin/settings/tiers`

### 주요 기능

#### 자동 등급 시스템
- **자동 등급 시스템**: ON/OFF 토글
- **자동 등급 승급**: 기준 금액 달성 시 자동 승급
- **자동 등급 강등**: 평가 기간 내 기준 미달 시 강등
- **평가 기간**: 월간/분기/연간/누적

| 등급 | 설정 항목 |
|------|------|
| **일반회원** | 기본 등급 (기준 금액 0원) |
| **우수회원** | 최소 구매 금액, 할인율, 적립률, 무료배송 |
| **VIP회원** | 최소 구매 금액, 할인율, 적립률, 무료배송 |

#### 등급 혜택 미리보기
설정한 내용이 표 형태로 미리보기됩니다.

## 15. 배송비 설정

### 접속 경로
`/admin/settings/shipping`

### 주요 기능

3가지 탭으로 구성됩니다.

#### 기본 설정 탭
| 항목 | 설명 |
|------|------|
| **기본 배송비** | 기본으로 적용되는 배송비 |
| **무료배송** | ON/OFF 토글 |
| **기준금액** | 무료배송 기준 금액 (이상 주문 시 무료) |

#### 구간별 탭
주문 금액에 따라 다른 배송비를 적용합니다.
- 구간 추가/삭제
- 각 구간별 최소금액, 최대금액, 배송비 설정
- 적용 예시 미리보기

#### 지역별 탭
지역에 따라 추가 배송비를 적용합니다.
- 지역별 추가 배송비 설정
